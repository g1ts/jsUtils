Null-safe access to properties (like as ?. in java)

  • function nsa(object, string)
  • object - root
  • string - path
Code: 
var nsa = window.jsUtils.nsa;
var o = {p1:{p2:{p3:1}}};
try{
  toLog(nsa(o,'p1.p2.p3'));
  toLog(o.p1.p2.p3);
  toLog(nsa(o,'p1.p3.p4'));
  toLog(o.p1.p3.p4);	// exception
}catch(e){toLog(e,'error: ')}

Random

  • function randomString(length, chars)
  • function randomHex(length)
  • function randomHexBytes(length)
  • length - length of random string (in randomHexBytes count bytes)
  • chars - array of characters used to generate string (default: a-z, A-Z, 0-9)
Code: 
toLog(window.jsUtils.randomString(16), 'randomString: ');
toLog(window.jsUtils.randomString(16,'123'), 'randomString: ');
toLog(window.jsUtils.randomHex(16), 'randomHex: ');
toLog(window.jsUtils.randomHexBytes(16), 'randomHexBytes: ');
